Using ES6+ Methods

The goal of this assignment is to use the latest ES6+ Javascript functionality.

  • Create at least four pairs of input boxes and submit buttons.
  • Use ES6+ array and object methods to manipulate the data input, and ouput the result with an explanation.
  • Use at least 4 ES6+ methods.
  • Remember it's best practice to add comments to your code.

Don't worry about validating data. Assume only numbers will be entered in the input box.

Optional Bonus: If you have completed the requirements above and are up for an extra challenge, you can take your project to the next level with following bonus activities.

  • Use string template literals for the output and use arrow functions.
  • Integrate two more ES6+ functions.

The reduce() method reduces the array to a single value. The return value of the function is stored in an accumulator (result/total).

The every() method tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value: True or False.

Array = [1, 2, 3, 4, 11, 14, 18, 20, 25, 30]




The filter() method returns an array containing elements of the parent array that match the set test.

Array = [1, 2, 3, 4, 11, 14, 18, 20, 25, 30]




The map() method creates a new array with the results of calling a function for every array element. The map() method calls the provided function once for each element in an array, in order.

Array = [1, 2, 3, 4, 11, 14, 18, 20, 25, 30]




The find() method creates a new array with the results of calling a function for every array element. The map() method calls the provided function once for each element in an array, in order.

Array = [1, 2, 3, 4, 11, 14, 18, 20, 25, 30]




The some() method checks if any of the elements in an array pass a test (provided as a function). If it finds an array element where the function returns a true value, some() returns true (and does not check the remaining values). Otherwise it returns false
